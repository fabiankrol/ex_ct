# ExCt

Mix task to execute common_test tests

## Installation

The package can be installed by adding `ex_ct` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_ct, "~> 0.0.0"}
  ]
end
```
